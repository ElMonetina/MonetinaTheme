# Monetina Theme

A collection of themes made by me:

- Focus: Based on Focus editor default theme.
- Gruber: Based on Tsoding's Gruber Darker theme for emacs.
- Gold: Monochrome Gold theme.
- Silver: Monochrome Silver theme.
- Diamond: Monochrome Diamond theme.
- Pearl: Monochrome Pearl theme.
- Yellow: Mostly monochrome yellow theme with blue accents.
- Blue: Mostly monochrome blue theme with yellow accents.
